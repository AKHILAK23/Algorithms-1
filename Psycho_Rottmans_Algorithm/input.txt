Note:
First we will enter the number of employees.
Then following that we will enter the weekly working hours of each employees line by line. 
Each entry represents the employee with employee number equal to the entry number.

Output Format

second largest weekly work hours
employee number

Note:
The first output should be the second largest weekly work hours.
Then in second line print the employee number of that employee with the corresponding working hours.
If there are more than one employee with the same wroking hours, we only take the first one.
